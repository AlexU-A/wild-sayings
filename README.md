# Wild Sayings

> **THIS IS SUPER IMPORTANT.** Possibly the most important advance in useless technology since the internet-connected toaster. Do not scroll past this. Your life may depend on it. (It won't, but a possum in a lab coat said it might, and who are we to argue?)

A Claude Code skill that generates hilarious, over-the-top hyperbolic sayings about any topic you throw at it. 22 humor traditions from around the world, from Southern drawl to Yiddish fatalism to Australian understatement to Swahili rhyming wisdom.

**[See it in action](https://alexu-a.github.io/wild-sayings/)**

## Install

```bash
npx skills add AlexU-A/wild-sayings
```

## Usage

```
/wild-sayings crazy
```

Output:
> That's crazier than a possum in a tuxedo trying to parallel park a riding mower in a Costco parking lot on Black Friday.
>
> Well, I've seen some crazy, but that's right up there with a squirrel day-trading cryptocurrency during a tornado while sitting in a lawn chair.
>
> Brother, that's got more crazy in it than a one-eyed alligator who just got appointed head of the HOA and is already sending violation letters to the swamp.

### More examples

```
/wild-sayings stubborn
/wild-sayings fast
/wild-sayings cheap
/wild-sayings hot
/wild-sayings confused
```

Say "hit me" or "more" after a result to get additional sayings on the same topic.

### Short mode

```
/wild-sayings crazy --short
```

Output:
> Crazier than a goat on a Roomba.
>
> More unhinged than a screen door in a tornado.
>
> That ain't crazy, that's a squirrel with a LinkedIn profile.

Bumper-sticker-length quips when you don't need the full campfire story.

## How it works

The skill draws from **22 humor traditions** spanning 6 continents, each with its own structural patterns, vocabulary, and comedic DNA:

| | | |
|---|---|---|
| Southern | Midwestern | Western/Cowboy |
| Modern/Universal | Yiddish | Scottish |
| Irish | Australian | Cajun/Creole |
| Appalachian | Texas | NYC/Brooklyn |
| Caribbean | Russian | Spanish/Latin American |
| German | Italian | French |
| Middle Eastern | East Asian | Indian/South Asian |
| African | | |

Each tradition brings unique patterns:
- **Southern** escalates animal chaos through layered imagery
- **Yiddish** turns God into a passive audience and wishes into curses
- **Scottish** compresses an entire insult into one untranslatable word
- **Australian** references deadly wildlife with casual understatement
- **Russian** finds freedom in having nothing left to lose
- **Caribbean** runs on Anansi trickster rhythm and patois musicality
- **Korean** hides untranslatable puns in everyday food metaphors
- **Cajun** turns bayou survival into social wisdom

The full dialect reference lives in [`DIALECTS.md`](DIALECTS.md).

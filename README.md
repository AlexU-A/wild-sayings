# Wild Sayings

A Claude Code skill that generates hilarious, over-the-top hyperbolic sayings about any topic you throw at it. Southern/country-style absurdist humor meets animal chaos.

## Install

```bash
npx skills add AlexU-A/wild-sayings
```

## Usage

```
/wild-sayings crazy
```

Output:
> That's crazier than a possum in a tuxedo trying to parallel park a riding mower in a Costco parking lot on Black Friday.
>
> Well, I've seen some crazy, but that's right up there with a squirrel day-trading cryptocurrency during a tornado while sitting in a lawn chair.
>
> Brother, that's got more crazy in it than a one-eyed alligator who just got appointed head of the HOA and is already sending violation letters to the swamp.

### More examples

```
/wild-sayings stubborn
/wild-sayings fast
/wild-sayings cheap
/wild-sayings hot
/wild-sayings confused
```

Say "hit me" or "more" after a result to get additional sayings on the same topic.

### Short mode

```
/wild-sayings crazy --short
```

Output:
> Crazier than a goat on a Roomba.
>
> More unhinged than a screen door in a tornado.
>
> That ain't crazy, that's a squirrel with a LinkedIn profile.

Bumper-sticker-length quips when you don't need the full campfire story.

## How it works

The skill instructs Claude to generate sayings using a formula:
- **Animal or creature** (raccoon, possum, armadillo, three-legged dog...)
- **Absurd human action** (filing taxes, applying for a mortgage, teaching yoga...)
- **Escalating modifier** (during a hurricane, in a Waffle House parking lot, at 3 AM...)

The humor comes from the collision of mundane human activities with animal chaos.
